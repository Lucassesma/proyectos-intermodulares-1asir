# 🗄️ Módulo: Gestión de Bases de Datos (sesmAI)

Este directorio contiene toda la implementación física y lógica de la base de datos para la consultora **sesmAI**.

## 📄 Documentación Incluida
* **Análisis de los datos.pdf**: Explicación del caso de uso y necesidades del cliente.
* **Modelo_relacional.drawio**: Diseño de la base de datos (tablas).
* **Scripts BBDD.pdf**: Código fuente completo (DDL, DML y DQL).
* **Tareas de administración.pdf**: Guía sobre seguridad, usuarios y backups.

## 🚀 Guía Rápida de Uso
Para levantar este entorno en local:
1. Importar el script SQL en **phpMyAdmin**.
2. Ejecutar las consultas de prueba incluidas en el PDF para verificar el funcionamiento de los `INNER JOIN`.
