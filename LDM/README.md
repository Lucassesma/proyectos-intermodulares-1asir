# Proyecto Intermodular - LDM

Este módulo contiene la gestión y transformación de datos mediante tecnologías XML.

## Contenido:
* **datos.xml**: Archivo de datos principal.
* **modelo.dtd**: Validación.
* **transform.xsl**: Hoja de transformación.
* **style.css**: Diseño.
* **datos.json**: Exportación JSON.
